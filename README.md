# Neural Network Fundamentals and Training Behavior Analysis

## Objective
The objective of this project is to build and analyze a simple feed-forward neural network for a supervised learning problem.

The project demonstrates:
- Forward propagation
- Loss calculation
- Backpropagation
- Weight updates
- Model evaluation
- Hyperparameter experimentation

---

## Dataset
Dataset used from the shared Google Drive folder.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- TensorFlow / Keras
